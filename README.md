# Physical Organic Chemistry Internship Repository

This repository contains the full workflow for the 5-week internship.

The goal of this repository is to reproduce a structured research pipeline.

## Structure

Each week has its own folder:
- week1_literature
- week2_ms_acquisition
- week3_data_analysis
- week4_computation
- week5_writing

These folders must be used in order. Do not skip ahead.

## Rules

- Do not modify any files outside `student_workspace/`
- Do not rename folders
- Do not change file structure
- Always follow the protocol files provided each week
